Text Sentiment Analyzer

Project Overview

This project analyzes the sentiment of a text file using Azure Cognitive Services' Text Analytics API. The goal is to leverage cloud computing technologies to understand the emotional tone of text content efficiently.

Why I Chose This Project

In today's world, understanding the sentiment of text data is crucial for various applications, from customer feedback analysis to social media monitoring. This project showcases how cloud computing and AI can simplify complex tasks, demonstrating my ability to work with cutting-edge technology.

Features

  • Automatically analyzes the sentiment of any text file (e.g., Positive, Neutral, Negative).
  • Provides detailed confidence scores for each sentiment category.
  • Breaks down sentiment analysis for individual sentences in the text.
  • Uses Azure Cognitive Services for scalability and reliability.

Technologies Used

  • Azure Cognitive Services: Text Analytics API
  • Programming Language: Node.js
  • JavaScript Libraries: Axios for API calls
  • Environment Variables: Managed securely using dotenv
  • HTML/CSS: For creating this webpage
